Assignment Goal

Design a slotting strategy that generalizes warehouse layout logic using a CSV grid of frequency data. SKUs should be sorted by pick frequency and assigned according to ABC classification (A = 20%, B = 30%, C = 50%).

Process Description

  1. Step 1: Load CSV file and parse pick frequency data.
  2. Step 2: Flatten the matrix row-wise to match SKUs (SKU0 = [0,0], SKU1 = [0,1], etc.).
  3. Step 3: Sort SKUs by pick frequency and assign ABC classes:
    • A: Top 20% of SKUs
    • B: Next 30%
    • C: Bottom 50%
  4. Step 4: Generate a new grid that places high-frequency SKUs closest to the dock (assumed at top-left).

Reflection

This task helped me understand how warehouse slotting strategies optimize operational efficiency by minimizing walking time for high-frequency SKUs. I learned to process grid-based frequency data and apply logic for spatial optimization in logistics design.
